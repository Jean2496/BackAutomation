Descripción

Este proyecto implementa un framework de QA Back Automation utilizando Rest Assured y Java.

Se automatizaron los siguientes escenarios:

• GET /usuarios - Listar usuarios
• POST /usuarios - Registrar usuario
• GET /usuarios/{_id} - Buscar usuario por ID
• PUT /usuarios/{_id} - Actualizar usuario
• DELETE /usuarios/{_id} - Eliminar usuario


Lenguaje: Java
Frameworks: Rest Assured, Cucumber
Arquitectura: Factory
Gestión de dependencias: Gradle

Instalación del proyecto

Clonar el repositorio: git clone https://github.com/Jean2496/BackAutomation.git
Ejecutar pruebas: ejecutar los escenarios desde el archivo .feature
